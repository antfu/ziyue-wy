# 子曰 ziyue, *Confucius said...*

This a library package for [wenyan-lang](https://github.com/LingDong-/wenyan-lang) (similar to cowsay)

## Install

Use [wyg](https://github.com/wenyan-lang/wyg) to install

```bash
wyg i 子曰
```

or

``` bash
wyg i ziyue
```

## Usage

```wy
吾嘗觀「「子曰」」之書。方悟「子曰」之義。

子曰「「巧言令色，鮮矣仁！」」。
```
